What is Dymatic?

Dymatic is a basic, open source, WIP 3D game engine created using C++. Currently, Microsoft Visual Studio is required to build, and only officially supports Windows. The engine includes a 3D rendering pipeline, physics, audio and scripting engine along with a wide array of editor tools.
